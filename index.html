Projects

Better Speedrun Timer

Better Speedrun Timer (or BST for short) is a speedrun retiming website that aims to be accurate and simple to use. It has a lot of features including:

  • Supports Youtube and Twitch
  • Supports Pauses
  • Frame based retiming
  • Customizable Mod Message
  • Customizable Default Framerate
  • Dark and light themes
  • Desktop and Moblie friendly UI
  • Open Source
  • French, Spanish, Italian, Portuguese and German Translations

Termux Rich Presence

Termux Rich Presence is a tool that brings Discord's Rich Presence to Mobile using Termux and NodeJS

RunGet

RunGet is a recreation of the original (now deleted) discord bot that posts new speedruns from speedrun.com. My Bot has the following features:

  • Customizable Runs Channel
  • Support for sending gamelists longer than Discord's Message Limit
  • Slash Commands
  • Usage of a Sqlite Database instead of JSON
  • The ability to add and delete games to the gamelist from full name and ID and addition to the abbreviation
  • The ability to bulk add and delete games from the gamelist
